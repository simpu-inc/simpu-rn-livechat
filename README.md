# simpu-rn-livechat

simpu react-native livechat sdk

## Installation

```sh
npm install simpu-rn-livechat
```

## Usage

```js
import SimpuLiveChat from 'simpu-rn-livechat';

// ...

const App = () => {
  return (
    <View>
      . ..
      <SimpuLiveChat />
    </View>
  );
};
```
